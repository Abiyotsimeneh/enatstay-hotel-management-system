## 🏨 About EnatStay

EnatStay represents warmth, care, comfort, and Ethiopian hospitality culture. The platform reflects welcoming hospitality and modern hotel operations with a premium SaaS dashboard experience.

## ✨ Features

- 📊 **Dashboard Analytics** - Real-time insights into hotel performance
- 🛏️ **Room Management** - Track and manage hotel rooms
- 📅 **Booking Management** - Handle guest reservations efficiently
- 👥 **Staff Management** - Organize hotel staff and schedules
- 📦 **Inventory Management** - Track supplies and stock levels
- 🔐 **Authentication** - Secure login and role-based access
- 📱 **Responsive Design** - Works seamlessly on all devices

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ installed
- MongoDB installed (for backend)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/enatstay-hotel-management-system.git
cd enatstay-hotel-management-system
```

2. Install client dependencies:
```bash
cd client
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:5173
```
